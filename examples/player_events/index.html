Registering and un-registering to player events

You can register and un-register to player events.
Register the the "kalturaPlayerReady" event to get a reference to the player.
You can then use the "kBind" and "kUnbind" API methods to register and un-register player events.
For a full list of available player events, look at the Kaltura API documentation.

In this example, the application listens to the player's "playerUpdatePlayhead" event and uses the event data to display the playback progress.
Clicking the "Remove Tracking Listener" button removes the listener and progress tracking stops.
A Flashvar object is used to invoke the player's autoPlay feature.
